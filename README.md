# gulpman
Create Modular Front-End Build System, based on gulp ,easy useage


####In Your gulpfile:

````Javascript
/**
 * Gulpfile.js
 */


var gulp = require('gulp'),
    gman = require('gulpman')


// your other tasks ...

```

####In Your CLI:

````Shell
# publish
gulp gm:publish

# develop and watch
gulp gm:develop

# clean
gulp gm:clean

# compile for develop,not watch
gulp gm:compile


```
